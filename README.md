1. `antlr YAPL.g4`
   * Se generaran los archivos relacionados al **lexer** y **parser** de antlr4.
2. `javac YAPL*.java`
   * Generacion de clases en java.
3. `grun YAPL program testIF.ypl -gui`
   * Para generar la interfaz grafica con el arbol de parseo.
